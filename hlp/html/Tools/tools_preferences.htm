Preferences command (Tools menu)

Use this command to change user preferences.

A property sheet dialog of 3 tabs is open.

File page:

"Temporary file location" edit box allows to specify where to place any temporary files the program creates.

"Use RAM for temporary files not larger than" edit box allows to specify what is size threshold for storing temporary data in memory, rather than on the disk.

"Max WAV file size" option allows to specify whether the program is allowed to create 4 GB WAV files. Some programs may have troubles with such size. Windows 98/ME doesn't support them at all, and the option is disabled for it.

"Save label/comment text with encoding" option chooses how the text data will be saved in WAV files.

  • 8 bit - the text will be saved in language-specific 8-bit encoding. If a file saved on one computer is read on another computer with different language settings, the labels and other text data may be shown incorrectly. This is the standard format used by most sound editing applications.
  • UTF-8 - the text will be saved using UTF-8 representation of UNICODE character set. Such representation doesn't depend on the language settings. Most languages are supported by default. This is a format extension and may not be supported in other applications.
  • UTF-16 - the text will be saved using UTF-16 representation of UNICODE character set. Such representation doesn't depend on the language settings. Most languages are supported by default. This is a format extension and may not be supported in other applications.

"Max file cache size" - specifies upper limit of memory allocated to cache the files. Default is 64 MB.

"Open files from command line and drag-and-drop as" option chooses how such files are opened:

  • Read only - the file(s) can only be viewed and listened to;
  • Direct edit - the file(s) will be opened for direct editing: any changes are saved immediately in the original copy;
  • Buffered - the file(s) will be opened in buffered mode: any changes will go to a temporary copy and commited to the permanent file only on File/Save command.

Undo/Redo page:

Undo group:

"Enable" - enables saving Undo information. This allows to return to the state before editing. This option can also be changed in More Undo/Redo submenu of Edit menu.

"Limit total size" - sets limit for the size of saved Undo information.

"Limit depth" - sets maximum number of Undo steps saved.

Redo group:

"Enable" - enables saving Redo information. This allows to repeat the undone operation. This option can also be changed in More Undo/Redo submenu of Edit menu.

"Limit total size" - sets limit for the size of saved Redo information.

"Limit depth" - sets maximum number of Redo steps saved.

"Remember current selection in Undo/Redo" - if checked, Undo/Redo operation will also restore the selection.

Sound page:

Playback group:

Device drop-down box selects an audio output device to use for playback. If "Microsoft Sound Mapper" is selected, a device selected in the system "Sounds audio devices" control panel applet will be used.

"Buffer size" and "Number of buffers" select depth of buffering. Some low-end systems may require more buffers for steady audio playback.

Recoprding group:

Device drop-down box selects an audio output device to use for recording. If "Microsoft Sound Mapper" is selected, a device selected in the system "Sounds audio devices" control panel applet will be used.

"Buffer size" and "Number of buffers" select depth of buffering. Some low-end systems may require more buffers for steady audio recording.

View page:

"Snap mouse selection to max sample" -  if checked, the mouse click will cause the current position go to the max sample of all that fit into the clicked pixel column.
