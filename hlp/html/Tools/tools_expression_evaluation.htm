Expression evaluation command (Tools menu)

Use this command to synthesize a waveform, using a mathematical formula, or apply a formula to the existing waveform.

The following dialog box is shown:

An expression can be typed directly into the "Expression" edit box, or a previously saved expression can be inserted in it.

Note that the expression result range should be from -1 to +1. This will be scaled to the full range of the wave file.

The expression obeys the usual rules of operation precedence.

The following operators can be used:

"+" - add two operands together;

"-" - subtract the right hand operand from the left hand one;

"/" - divide the left hand operand by the right hand one;

"*" - multiply the two operands

"%" - the left hand operand modulo right hand operand. In other words, the remainder of dividing the left operand by the right one.

"&" - bitwise AND of two operands. The operands are converted to integers first.

"|" - bitwise OR of two operands. The operands are converted to integers first.

"^" - bitwise XOR of two operands. The operands are converted to integers first.

"~" - bitwise one's complement of an operand. The operand is converted to an integer first.

The following functions can be used:

"sin(x)" - sine, x is in radians;

"cos(x)" - cosine, x is in radians;

"tan(x)" - tangent, x is in radians;

"exp(x)" - exponent (e power x);

"exp10(x)" - decimal exponent (10 power x);

"sqrt(x)" - square root of x;

"int(x)" - convert x to integer. If x is a floating point number, it is rounded toward zero;

"sinh(x)" - hyperbolic sine;

"cosh(x)" - hyperbolic cosine;

"tanh(x)" - hyperbolic tangent;

"ln(x)" - natural logarithm of x(base e);

"log10(x)" - decimal logarithm of x (base 10);

"abs(x)" - if x is negative, negate it;

"pow(x, y)" - make x power y;

Operands tab shows chat you can use in the expression:

Operands have the following meaning:

"noise()" - white noise, evenly distributed from -1 to 1, mean square root is sqrt(1/3);

"T" - time in seconds from the beginning of file;

"N" - number of sample from the beginning of the file;

"DT" - length of the file, in seconds;

"DN" - number of samples in the file;

"F" - sample rate;

"pi" - 3.1415926......;

"t" - time in seconds from the beginning of selection;

"n" - number of samples from the beginning of selection;

"dt" - length of the selection, in seconds;

"dn" - length of the selection, in samples;

"wave" - current wave sample of the file;

"Saved Expressions" tab shows previously saved expressions, including those provided as default.

The expressions can be grouped into categories.

"Group" drop-down list selects a group. There are two predefined groups: "Waves" and "Noises". "All Expressions" selection combines them all.

The lower drop-down list selects an expression from the group. After you've found the one you need, press "Insert" button, to put it into the expression edit box.

"Delete" button deletes the selected expression.

"Export" button saves all the expressions in a file with .expr extension. "Import" button loads expressions from a previously saved file.

"Save As..." button saves the contents of the expression edit box with a name and group you choose. The following dialog is opened:

Type the expression name into "Name" edit box, or select it from a drop-down list.

Type the group name into "Group" edit box, or select it from a drop-down list.

Type the comment into "Comment" edit box. Press OK to save the expression, or Cancel.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.

You can expand the file, if the selection length goes beyond the file end.
