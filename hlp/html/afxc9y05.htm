Save command (File menu)

Use this command to save the active sound to its current name and directory. When you save a sound file for the first time, WaveSoap displays the Save As dialog box so you can name your sound file. If you want to change the name and directory of an existing sound file before you save it, choose the Save As command. You can also save a copy of the open file, by selecting Save Copy command.

Shortcuts

Toolbar:

Keys:CTRL+S
