Waveform view

This part of the WaveSoap window shows the sound data as waveform. It can be shown with different scale - the more sound fits in the screen the less details is displayed. Current scale on the time axis is shown in the status bar as 1:n, for example 1:2048. It means that n samples are shown in one pixel column. Scale is always power of 2.

The selected part of the sound is shown with black background. 

You can change horizontal scale by using menu View commands Zoom In Time (magnify the wave) and Zoom Out Time (diminish the wave). For Zoom In Time you can also use "gray +" key (on the keypad) or toolbar button. For Zoom Out Time you can also use "gray -" key (on the keypad) or toolbar button. You can set maximum "Zoom Out", to make all the sound data fit to the window, by clicking toolbar button.

Another type of sound presentation is FFT spectrum view. To switch to it, use FFT Spectrum command of View menu, or Atl+2 keyboard shortcut, or toolbar button.

To switch back to waveform view from FFT spectrum view, use Waveform command of View menu, or Atl+1 keyboard shortcut, or toolbar button.
