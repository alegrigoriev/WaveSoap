View menu commands

The View menu offers the following commands:

Toolbar   Shows or hides the toolbar.
Status Bar   Shows or hides the status bar.
Waveform Alt+1 Shows waveform
FFT Spectrum Alt+2 Shows FFT spectrum instead of waveform
Spectrum Section Alt+3 Shows or hides spectrum section view
Outline Alt+4 Shows or hides the file outline
Zoom In Time Num + Magnifies the file view on the time axis
Zoom Out Time Num -
Zoom To Selection Shift+Num+ Changes view scale so all the selected area would fit to the window
     
Time Ruler   Opens submenu for enabling or disabling time ruler and changing time display format
Zoom In Amplitude Alt+Up Magnifies the waveform view on the amplitude axis
Zoom Out Amplitude Alt+Down
Amplitude Ruler   Opens submenu for enabling or disabling amplitude ruler and changing amplitude display format
Rescan Peaks   Rescan the sound file to collect information about peak outline.

Time ruler popup menu:

Enable Alt+5 Enables or disables time ruler display
Samples   Show sample number in the time ruler.
HH:MM:SS   Show time as hour:minute:second.milisecond in the time ruler.
Seconds   Show time as second.milisecond in the time ruler, for example 550.435 - 550 seconds 435 miliseconds.
HH:MM:SS.FF   Show time as hour:minute:second.CD_frame in the time ruler. CD frame rate is 75 per second. For example 1:02:35.15 - 1 hour 2 minutes 35 seconds and 15 frames (15/75 second).

Amplitude ruler popup menu:

Enable Alt+6 Enables or disables amplitude ruler display
Sample value   Show sample value on the amplitude ruler.
Percent   Show percents of the full amplitude on the ruler.
dB   Show decibel values on the ruler.
