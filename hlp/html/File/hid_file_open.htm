Open command (File menu)

Use this command to load an existing sound file in a new window.

In Windows XP, the command shows the following dialog:

 

In Windows 2000, Windows ME, Windows 98, the dialog is different:

Recent drop-down box allows you to select one of your recently used folders.

You can select and open more that one file at one time. To select multiple files, hold Ctrl key while clicking the files. To select a group of files, click the first file, then press Shift key and click the last file.

To open the selected file, click Open button, or just double click on the file in the list.

Files of Type drop-down box allows to select the type of file you want to open:

All supported types: files of all types supported by the program will be shown in the directories.
Windows sound files (*.wav): only WAV-files will be shown
MPEG audio files (*.mp3): only MP3 files will be shown
Windows media files (*.wma): only WMA files will be shown
All Windows Media files (*.wma, *.wmv, *.asf): all files in Windows Media format are shown
Raw (unformatted) sound files (*.raw): RAW files will be shown

If Open as read-only checkbox is set, the file(s) will be opened for reading only; any changes won't be allowed.

If Operate directly on the sound file checkbox is set, all edit operations will be performed directly over the file itself, rather than over its temporary copy. This feature is not available in the demo version.

You can resize the dialog, by clicking and dragging its border or the resize handle .

You can create new sound with the New command.

Note: In Windows 98, the "Places" bar on the left side of the dialog is not shown.

You can open multiple files at once. Use the Window menu or Ctrl+Tab keyboard shortcut to switch among the multiple open sound files. See Window 1, 2, ... command.
