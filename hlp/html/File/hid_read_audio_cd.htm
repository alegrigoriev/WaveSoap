Read Audio CD Tracks command (File menu)

Use this command to convert audio CD tracks to a sound file. You can save it as WAV, or MP3, or WMA file.

The command opens the following dialog:

If you have more than one CD drive, you can select one with Drive drop-down list.

Reading speed can be selected with Extraction speed drop-down list. Some low-quality CD drives cannot perform full error correction procedures while reading with maximum speed. If you experience data errors (seen as spikes in the read audio file), reduce the speed down to 8x or 2x. In Windows XP, you only have this option if Adaptec's ASPI is installed, or you're running with Administrator rights (which is not generally recommended due to security concerns).

Tracks list shows all audio tracks, their names and length. To edit the name, select it and press F2 (or click with the mouse on it). Checkboxes select what tracks will be processed.

The CD tracks can be saved as uncompressed WAV files, or Windows Media Audio (WMA), or as MPEG-1 Layer III (MP3).

The files can be saved immediately, or just opened in the editor.

For WMA and MP3 files, "Format" drop-down list allows to select the files' bit rate (measured in kilobits per second - kbps). The higher the bit rate, the better is sound quality, but the files take more space.

The program supports LAME MPEG encoder, as well as ACM driver by Fraunhofer Institute.
