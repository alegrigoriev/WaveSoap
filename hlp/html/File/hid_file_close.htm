Close command (File menu)

Use this command to close all windows containing the active sound file. WaveSoap suggests that you save changes to your sound file before you close it. If you close a file without saving, you may lose all changes made since the last time you saved it, unless the file is opened in the direct mode. Before closing an untitled sound file, WaveSoap displays the Save As dialog box and suggests that you name and save the document.

Shortcuts:

Mouse: double click on the Close icon on the file's window or clicking the Close (X) button, as shown below

Keys: Ctrl+Shift+F4

Note: Ctrl+F4 key combination will also close the sound file, if it is open in one window only. If you opened secondary view to the file, using Window/New Window menu command, Ctrl+F4 closes only one window, while Ctrl+Shift+F4 closes them all.
