Split To Files command (File menu)

Use this command to save different regions of the active sound file into separate files. By default, the program splits the original file by the markers and regions and give each file the marker or region name. You can add a prefix to each file name.

You can save the data into WAV, MP3 and WMA files.

The file list shows the base names, the segment begin, end and length. The list is pre-filled with markers and region names. To edit the name, select it and press F2 key, or click on it one more time. The name is also saved as the file title.

To add a segment to the list, click "New" button, or press Insert key while in the list.

To delete a segment from the list, click "Delete" button, or press Delete key.

You can edit the selected segment's start and end positions, or length. See Specifying position and selection for Time Format explanation.

"Filename prefix" edit field allows to add some text and an incrementing number to the file names. It can consist of arbitrary text and number formatting sequences, starting with percent sign:

%d - segment number, zero-based. First segment gets number zero, second gets number one, etc.

%nd - segment number, with the start specified. For example, %2d - start numbering from 2: 2, 3, 4, 5...

%0nd - segment number, with the start specified, padded with zeros. For example, %01d gives the following sequence: 01, 02, 03, and so on. %005d gives: 005, 006, 007, and so on.

%% - insert a single percent sign to the file name.

Enter the target directory name "Save to" edit field, or select one of the previous names from the drop-down list, or click [...] button to browse for the directory.

"Save as type" drop-down list allows to select the target file type: Windows sound files (WAV), MPEG audio files (MP3), Windows Media files (WMA).

For WAV files, "Format" drop-down list allows to select the target encoding. PCM stands for uncompressed file.

"Attributes" drop-down list selects one of available target formats for the selected endoding.

If "Compatible formats only" box is checked, only the formats with the same number of channels and sample rate will be shown in "Format" and "Attributes" lists.

For MP3 files, "Encoder" drop-down list will be shown with the encoders available.
