New command (File menu)

Use this command to create a new blank sound file in Wave Soap. 

The program shows a dialog, which allows you to specify the file attributes and its initial length:

Initial length is specified in hour:minutes:seconds.fraction format. For example: 1:0:0 - 1 hour; 100:30.5 - 100 minutes 30.5 seconds. You can specify zero length. In this case, you can extend the file later, by using Edit/Insert Silence command, or by pasting the data from the clipboard.

Sampling rate can be from 1 to 1000000 samples per second. You can also select one of the standard rates from the drop-down list. Most of the time, 44100 samples per second sample rate is used, which is CD-quality sound.

If Show this dialog only if Shift pressed checkbox is set, the dialog will be shown only if you hold Shift while selecting the menu command or toolbar button, or if you use Ctrl+Shift+N keyboard shortcut.

You can also create a new file by pasting data from clipboard or by reading from audio CD tracks.

You can open an existing sound file with the Open command.
