1, 2, 3, 4 command (File menu)

Use the numbers and filenames listed at the bottom of the File menu to open the last files you closed. Choose the number that corresponds with the sound file you want to open.

After the file name, the menu may show (D) or (RO).

(D) means the file was opened in direct mode. If you select that file name in the menu, it will be opened again in direct mode.

(RO) means the file was opened in read-only mode. If you select that file name in the menu, it will be opened again in read-only mode.
