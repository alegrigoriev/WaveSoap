Save Copy As command (File menu)

Use this command to save a copy of the active sound file under different name and/or directory and/or file type. WaveSoap displays the Save As dialog box.

To save a sound with its existing name and directory, use the Save command.

After the copy has been saved, the program will ask if you want to open the saved file.

Shortcuts

Toolbar:

Keys: Shift+F12
