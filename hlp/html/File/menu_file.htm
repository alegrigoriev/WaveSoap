File menu commands

The File menu offers the following commands:

New   Opens New submenu
Open Ctrl+O Opens an existing sound file.
Save   Opens Save submenu
Read Audio CD Tracks...   Opens CD Audio Extraction dialog
Close Ctrl+Shift+F4 Closes an opened sound file.
Properties...   Displays and allows to change sound properties
Statistics… Calculates and displays file statistics
Exit Alt+F4 Exits WaveSoap application.
 

New item opens the following submenu:

Blank sound Ctrl+N Creates an empty sound file.
New From Clipboard Ctrl+E Creates a new sound file and inserts the current clipboard contents

Save item opens the following submenu:

Save Ctrl+S Saves an opened sound file using the same file name.
Save As... F12 Saves an opened sound file to a specified file name.
Save Copy As... Shift+F12 Saves an opened sound file to a specified file name.
Save Selection As... Ctrl+Shift+F12 Saves the selected area into a specified file name.
Split to files...   Saves as the multiple files, divided by markers and regions
Save All Ctrl+Shift+S Save all modified sound files
