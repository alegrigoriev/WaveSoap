Statistics command (File menu)

Use this command to show the sound statistics, such as amplitude and positions of the sound peaks, RMS (mean square root), average value (DC component) and frequency of zero crossings.

Statistics are calculated for the current selection, or, if the selection is empty, for the whole file.

You can also move the cursor to the position of the biggest sound peak.

The following items are shown:

  • Cursor position - current position of the cursor in the file;
  • Sample value at cursor;
  • Maximum negative peak positions and values;
  • RMS power - square root of sum of all squared samples. It is average power of the file or selection.
  • Average value (DC offset). You can check it to see if the file requires DC adjustment;
  • Average frequency of zero crossings;
  • CRC - cyclic redundancy check value. you can use it to see if two files or selections contain identical sound;
  • Checksum - value equivalent to the checksum calculated by Audiograbber program (http://www.audiograbber.com-us.net).

Go to max sample button moves the current position to the biggest peak (either negative or positive).

OK button closes the dialog.
