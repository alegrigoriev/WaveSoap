Save command (File menu)

Use this command to save the active sound file to its current name and directory. When you save a new sound file for the first time, WaveSoap displays the Save As dialog box, so you can name your sound file.

If you want to change the name and directory of an existing sound file before you save it, choose the Save As command. Use this command also to save the file with different compression or sample rate.

You can also save a copy of the open file, by selecting Save Copy command. The original file will remain open.

After the file has been saved, the application reloads it. It may prompt you how you want the file to be reopen:

  • If the uncompressed file was originally opened in buffered mode, you have options to reopen in buffered mode again (this will create a new temporary file), reopen it in direct mode to work immediately on the file, or close it altogether.
  • If the file was saved with compression, you have options to reload it from the saved copy, or continue working with the current data. Most compression formats have compression loss, and reloaded data may be different from original.
