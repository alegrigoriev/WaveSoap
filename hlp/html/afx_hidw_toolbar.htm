Toolbar

The toolbar is displayed across the top of the application window, below the menu bar. The toolbar provides quick mouse access to many tools used in WaveSoap.

To hide or display the Toolbar, choose Toolbar from the View menu (ALT, V, T).

ClickTo:

Create a new sound file. WaveSoap may display a dialog to specify the new file parameters

Open an existing sound file. WaveSoap displays the Open dialog box, in which you can locate and open the desired file.

Save the active file with its current name. If you have not named the document, WaveSoap displays the Save As dialog box.

Print the active document.

Cut: Remove selected sound data from the file and stores it on the clipboard.

Copy the selection to the clipboard.

Paste: Insert the contents of the clipboard at the insertion point.

Reverse the last editing (Undo).

Redo the last editing after Undo.
