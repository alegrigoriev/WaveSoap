Edit menu commands

The Edit menu offers the following commands:

Undo Reverse previous editing operation.
  Redo Redoes previously undone operation
Cut Deletes selected sound data from the file and moves it to the clipboard.
Copy Copies selected sound data from the file to the clipboard.
Paste Inserts data from the clipboard into the file.
Stop Stops current lengthy operation
Go To… Goes to the specified position in the sound file
Paste As new File Creates a new file and inserts sound data from the clipboard in it
Selection… Sets selection boundaries
Delete Deletes selected sound data from the file
