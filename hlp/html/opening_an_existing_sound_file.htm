Opening an existing sound file

The program supports standard WAV files with uncompressed PCM data as well as compressed sound data. It also supports MP3 encoded files and Windows Media files. The files with unsupported internal structure can be opened as raw sound files.

The files can be opened for editing in direct or buffered mode, and also in read-only mode.

When a file is opened in direct mode, all editing operations change the file itself. Only uncompressed WAV files with 16-bit data can be opened in direct mode. This mode saves some little time when opening a file.

In buffered mode, though, a temporary file is created, which holds all the new contents of the file. As you open the 16 bit PCM sound file, it's content is not copied immediately to the temporary file. It is copied while you are browsing and editing it. The temporary file is usually created in the same folder, where the original file is located, to speed file save operation. 8 bit PCM and compressed files, such as compressed WAV, MP3 and WMA, take some more time to open, because WaveSoap needs to convert them to 16 bit PCM format.

Files can also be opened as read only. WaveSoap won't allow to make any changes in such files.
