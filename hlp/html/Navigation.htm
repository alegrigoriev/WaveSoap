How to navigate in the sound file

There are various means to move around in the sound file.

The current edit point is shown as a blinking vertical line, called caret, like one in a text editor. The caret may cover both channels of a stereo file, or one of them.

You can use arrow keys to move the edit point by one pixel to the left or right.

You can click in the waveform view, to set the edit point wherever you want.

You can use Edit/Go To command to move to a position specified in time units (Shortcut: Ctrl+G).

You can click in the outline bar, to move to the corresponding position in the file. If Snap mouse position to selection option is enabled in the Preferences dialog, the edit point will be set to the peak sample position. If you click on a marker or a region boundary (depicted as a dotted line), the edit point will be set to it.

Ctrl+Left Arrow moves the edit point to the next marker or region boundary, or to the end of file.

Ctrl+Right Arrow moves the edit point to the previous marker or region boundary, or to the beginning of file.

Home key moves the edit point to the beginning of the current selection.

End key moves the edit point to the end of the current selection.

Ctrl+Home key moves the edit point to the beginning of file.

Ctrl+End key moves the edit point to the end of file.

Page Up and Page Down  keys move the edit point correspondingly backward and forward, by a screen worth of sound.

Note than when Shift key is held, most navigation keys will extend the selection. For example, Ctrl+Shift+End combination selects from the current position to the end of file. Mouse click with Shift held selects to the corresponding position.
