How to navigate in the sound file

You can use various means to move around in the sound file.

The current edit point is shown as a blinking vertical line, called caret, like one in a text editor. The caret may cover both channels of a stereo file, or one of them.

You can use arrow keys to move the caret by one pixel to the left or right.

Ctrl+Left Arrow moves the caret to the next marker or region boundary, or to the end of file.

Ctrl+Right Arrow moves the caret to the previous marker or region boundary, or to the beginning of file.

Begin key moves the caret to the beginning of the current selection.

End key moves the caret to the end of the current selection.

Ctrl+Begin key moves the caret to the beginning of file.

Ctrl+End key moves the caret to the end of file.

You can use Edit/Go To command to move to a position specified in time units (Shortcut: Ctrl+G).

You can click in the outline bar, to move to some position in the file. The cursor will be set to the peak sample position, if Snap mouse position to selection option is enabled in the Preferences dialog.
