Status Bar

The status bar is displayed at the bottom of the WaveSoap window. To display or hide the status bar, use the Status Bar command in the View menu.

The left area of the status bar describes actions of menu items as you use the arrow keys to navigate through menus. This area similarly shows messages that describe the actions of toolbar buttons as you depress them, before releasing them. If after viewing the description of the toolbar button command you wish not to execute the command, then release the mouse button while the pointer is off the toolbar button.

When no menu item or toolbar button is selected, the left area shows status of the currently executing operation.

Three boxes on the right hand side of the status bar indicate parameters of the currently active sound file:

  • Sampling rate;
  • Stereo or Mono;
  • File duration.

If you click the mouse right button on the sampling rate box, a Sample Rate popup menu will be shown, to allow you to change the file playback rate, or resample the file.

If you click the mouse right button on the Storeo/Mono box, a Channels change popup menu will be shown, to allow you to change the file from stereo to mono, or vice versa.

If you click the mouse right button on the file duration box, a time mode popup menu will be shown, to allow you to change the time display mode.
