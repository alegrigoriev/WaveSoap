Spectrum section pane

Spectrum section pane can be shown using View/Spectrum section command, or with Alt+4 keyboard shortcut.

In this side window you can see detailed spectrum on the current edit point (where caret is shown). If there is non-empty selection, the spectrum is averaged over it and will be more smooth.

On the left hand side, there is frequency scale, with two buttons on its lower end, for zooming it in and out.

Above the view, there is a decibel scale for the signal power in each frequency band. On its left end, you can see two buttons to zoom it in and out.

Right click in the view opens the pop-up menu with the following commands:

"Zoom In/ZoomOut" - increase/decrease scale on the decibels (horizontal) axis;

"Zoom In Frequency/Zoom Out Frequency" - increase/decrease scale on the frequency (vertical) axis;

"Number of FFT bands" - allows to select frequency resolution. It is synchronized with spectrum view.

"FFT window type" - allows to select different windowing function.

"Show Noise Masking" - turns on and off display of noise masking function and threshold.

"Hide" - hides this pane. You can reopen it with Alt+4 keyboard combination.

"Show crosshair cursor" - toggles display of vertical and horizontal dotted lines that follow the mouse cursor.
