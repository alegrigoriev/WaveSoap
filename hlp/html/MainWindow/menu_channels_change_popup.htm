Stereo/Mono popup menu

This menu appears when you click on the status bar's Mono/Stereo box.

  • If the file is mono, and you select "Stereo", the following confirmation dialog appears:



    If you click "Yes" button, the file will become stereo. The mono sound will be copied to both left and right channels of the new stereo file.
     
  • If the file is stereo, and you select "Mono", the following dialog appears:


    You will need to choose, what data will go to the converted file. In most cases you want to preserve both left and right channels, so Left+Right will be your choice.
