Copy command (Edit menu)

Use this command to copy selected sound data onto the clipboard. This command is unavailable if there is no data currently selected.

Copying sound data to the clipboard replaces the contents previously stored there.

The stored data can be inserted to this or another open sound file, by performing Edit Paste command.

The stored data can also be used to create a new sound file, by selecting Edit/Paste As New File command.

Shortcuts:

Toolbar:

Keys:Ctrl+C
