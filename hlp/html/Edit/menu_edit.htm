Edit menu commands

The Edit menu offers the following commands:

Undo Ctrl+Z Reverse previous editing operation.
Redo Ctrl+Y Redoes previously undone operation
More Undo/Redo   Opens submenu with commands to enable and disable Undo/Redo and set options
Copy Ctrl+C Copies selected sound data from the file to the clipboard.
Cut Ctrl+X Deletes selected sound data from the file and moves it to the clipboard.
Delete Del Deletes selected sound data from the file
Paste Ctrl+V Inserts data from the clipboard into the file.
Paste As new File Ctrl+E Creates a new file and inserts sound data from the clipboard in it
Insert Silence   Inserts specified amount of blank space (silence)
Fade   Opens submenu with In and Out commands
Stop Esc Stops current lengthy operation
Go To… Ctrl+G Goes to the specified position in the sound file
Select All Ctrl+A Select all file data
Selection… Ctrl+D Sets selection boundaries
Marker/Region... M Allows to set or edit a marker or region
Lock Channels   Locks both stereo channels together, so all operations will change them the same way
 

Undo/Redo submenu:

Enable Undo Toggles Undo enable/disable
Enable Redo Toggles Redo enable/disable
Undo all changes TODO
Clear Undo Data Deletes all previous Undo data
Clear Redo Data Deletes all previous Redo data
Undo/Redo Options... Sets Undo/Redo options, such as maximum depth and size, and shows the current size of saved Undo/Redo data.

Fade In/Out submenu:

In Fades the sound in from silence to full volume
Out Fades the sound out from full volume to silence
