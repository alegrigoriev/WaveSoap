Marker/Region command (Edit menu)

Use this command to create or modify a marker or region.

The following dialog is shown:

Name edit box allows to enter or change marker/region name.

Type buttons select whether this is a marker (no length) or a region (with length).

Start time edit box allows to specify the marker's position or the region's begin.

End and Length time edit boxes allow to enter end time or length of the region.

Time format drop-down list allows to select different kinds of time representation.

Clicking on Delete button deletes the marker or region. Only the record is deleted; the sound data remains unchanged.

Shortcuts:

Keys: M
