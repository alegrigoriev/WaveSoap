Insert Silence command (Edit menu)

Use this command to add blank space to the sound file.

If there is non-empty selection, the following dialog is shown first:

If you click Mute, the selected area will be cleared, as if  Process/Mute command is executed.

A dialog is opened, to select blank space length and position:

Length and Position specify how much and where the silence will be inserted. For time format description, see Specifying position and selection.

Channel selects where the silence will be inserted. If you select one channel in the stereo file, the rest of the sound will be moved and synchronization between channels will be lost. The following warning will be shown:

Use it only if you need special processing.
