Paste command (Edit menu)

Use this command to insert a copy of the clipboard contents at the insertion point. This command is unavailable if the clipboard is empty.

If the clipboard data has different sample rate that the target file, WaveSoap asks:

Resample the clipboard contents on paste - the data will be converted to the target sample rate.

Paste sound samples as is, without resampling - the data will be inserted as if the sample rate is the same.

If the data stored in the clipboard has more channels than the target file or selection, WaveSoap asks what to do:

If there is a non-empty selection, WaveSoap asks:

Replace selection with the clipboard contents  - the clipboard selection will replace the selection, shrinking or expanding the file, as necessary.

Paste as much as selection length - no more than selected will be pasted. The file length will not change.

Insert clipboard data at current caret position - just insert the clipboard at the current edit point. The file will be expanded.

NOTE: If only one channel of the stereo file is selected, inserting the clipboard contents will cause offset between channels. If you don't want that, make sure both channels are selected, or Lock Channels button is checked, like .
