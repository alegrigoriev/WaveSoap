Go To command (Edit menu)

Use this command to move the edit point (caret) to the desired position in the file. You can specify time, or sample number, or a marker or region.

WaveSoap opens the following dialog:

Position edit box allows to type a desired time position, or sample number, depending on selection in Time Format drop-down box. The dropdown list contains positions of markers and regions.

See Specifying position and selection for more information.

Shortcuts:

Key: Ctrl+G
