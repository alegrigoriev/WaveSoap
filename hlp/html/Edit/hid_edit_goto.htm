Go To command (Edit menu)

Use this command to move the edit point (caret) to some exact position in the file. You can specify time, or sample number, or a marker or region.

WaveSoap open s the dialog:

Position edit box with a dropdown list allows to type a desired time position, or sample number, depending on selection in Time Format drop-down box. See Specifying position and selection for more information.

Shortcuts:

Key: Ctrl+G
