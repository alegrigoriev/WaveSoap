Undo/Redo Options command (Edit menu)

This command opens the following dialog box:

Undo group:

Text shows the current amount of data saved for Undo, and number of steps that can be undone.

"Clear" button removes all saved Undo data.

"Enable" - enables saving Undo information. This allows to return to the state before editing. This option can also be changed in More Undo/Redo submenu of Edit menu.

"Limit total size" - sets limit for the size of saved Undo information.

"Limit depth" - sets maximum number of Undo steps saved.

Redo group:

Text shows the current amount of data saved for Redo, and number of steps that can be undone.

"Clear" button removes all saved Redo data.

"Enable" - enables saving Redo information. This allows to repeat the undone operation. This option can also be changed in More Undo/Redo submenu of Edit menu.

"Limit total size" - sets limit for the size of saved Redo information.

"Limit depth" - sets maximum number of Redo steps saved.

"Remember current selection in Undo/Redo" - if checked, Undo/Redo operation will also restore the selection.
