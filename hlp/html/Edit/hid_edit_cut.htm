Cut command (Edit menu)

Use this command to remove the currently selected sound data from the file and put it on the clipboard. This command is unavailable if there is no data currently selected or the file is open as read-only.

Cutting sound data to the clipboard replaces the contents previously stored there.

The stored data can be inserted to this or another open sound file, by performing Edit Paste command.

The stored data can also be used to create a new sound file, by selecting Edit/Paste As New File command.

NOTE: If only one channel of the stereo file is selected, cutting a selection will cause offset between channels. If you don't want that, make sure both channels are selected, or Lock Channels is checked.

Shortcuts:

Toolbar:

Keys:Ctrl+X
