Markers and Regions

Every so often, you may need to "bookmark" some place or some range in the file.

You can set a marker (known also as cue mark) on a point in the file, or assign a selection as a region.

Markers and region boundaries are shown as vertical dotted lines in the waveform, spectrum view, and in the file outline bar.

Region boundaries are shown on the time ruler with cue signs like these: and . The first marks the region begin, the second marks the end.

You can drag the cue signs with mouse.

Ctrl+Left Arrow and Ctrl+Right Arrow key combinations move the edit point (caret) to the previous or next marker or region boundary.

Double click in the waveform or spectrum view selects data between two adjacent markers or region boundaries. The same for double click in the wave outline bar.

Right click on the cue sign brings a menu:

Cursor Here - moves the edit point (caret) to the cue position.

Select (region only) - selects this region.

Delete - deletes the marker or region. Sound data is not changed.

Move to Cursor - moves this cue point to the current edit point position.

Edit... - opens Marker/Region dialog. The same can be done with keyboard shortcut: M.

All operations with markers and regions can be undone/redone.
