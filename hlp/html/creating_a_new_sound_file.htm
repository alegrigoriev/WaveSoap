Creating a new sound file

New sound files can be created either as blank files of some length (length can be zero), or can be imported from an audio CD track. You can also record a new file from an analog source, such as cassette or LP player.

To create a blank file, use New command of File menu, or Ctrl+N shortcut, or click toolbar button.

You can also create a new file from the clipboard contents, by using New From Clipboard command of File menu (when no file is open), or Paste As New File command of Edit menu, or Ctrl+E shortcut.

To import one or more tracks from an audio CD, use Read Audio CD Tracks command of File menu.

To record from an analog source, use Record command.

You can use Expression Evaluation command from Tools menu, to fill a blank file with a calculated waveform.

You can add more data to a file from the clipboard, by using Paste command.

You can use Insert Silence command from Edit menu to add more blank space to a file.
