File menu commands

The File menu offers the following commands:

New Creates an empty sound file.
New From Clipboard Creates a new sound file and inserts clipboard contents
Open Opens an existing sound file.
Close Closes an opened sound file.
Save Saves an opened sound file using the same file name.
Save As Saves an opened sound file to a specified file name.
Save Copy Saves an opened sound file to a specified file name.
Save All Save all modified sound files
Properties Displays and allows to change sound properties
Print Prints a document.
Print Preview Displays the document on the screen as it would appear printed.
Print Setup Selects a printer and printer connection.
Statistics… Calculates ans displays file statistics
  Exit Exits WaveSoap application.
