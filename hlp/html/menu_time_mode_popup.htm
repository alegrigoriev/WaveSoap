Time display mode popup menu

This menu allows you to change the time display mode. The available options are:

  • samples - file offset and length is shown in sound samples;
  • HH:MM:SS - file offset and length is shown in Hours:Minutes:Seconds.miliseconds
  • Seconds - file offset and length is shown in Seconds.miliseconds
  • HH:MM:SS.FF - file offset and length is shown in Hours:Minutes:Seconds.CD_frames, assuming 75 frames (558 samples each) per seconds.
