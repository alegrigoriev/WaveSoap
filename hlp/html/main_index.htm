Commands:

File menu - provides commands for opening and saving audio files, and also for capturing sound data from CD or analog source;

Edit menu - provides basic commands for sound manipulation, such as selecting, copying, deleting and inserting sound data;

View menu - provides commands to control the data visualization, such as zooming and other;

Process menu - provides commands for special processing, such as cleanup, level normalization, volume and playback rate change, etc;

Tools menu - provides additional tools for advanced operations, such as processing with expression evaluation.

Record menu

Window menu - allows to manipulate document menus;

Help menu - provides access to the help index and topics.
