File Save As dialog box

The following options allow you to specify the name and location of the file you're about to save:

File Name

Type a new filename to save a file with a different name. WaveSoap adds the extension you select in the Save As Type box.

Save as Type

Select the type of file you want to save:

Windows sound files (*.wav): save as a sound file in WAV format;
MPEG audio files (*.mp3): Save the file compressed in MPEG-I Layer 3 (MP3) format;
Windows media files (*.wma): Save the file compressed in WMA format;
Raw (unformatted) sound files (*.raw): Save the file as an unformatted RAW file.

Save in:

Select the directory in which you want WaveSoap to save the file.
