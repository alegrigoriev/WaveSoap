Fade In/Out dialog

This dialog appears during some operations, to allow smooth transition between selection and the rest of the file.

If "Enable fade in/out" box is checked, WaveSoap application will ensure smooth transition between selection and the rest of the file.

"Transition length" edit box specifies it in milliseconds. The allowed range is from 1 to 2000.
