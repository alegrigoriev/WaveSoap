Selection dialog

Use this dialog to select an exact area for operation.

"Selection" drop-down list allows to choose one of predefined ranges, such as "All file", "From begin to current position", "From current position to end", or one of regions.

Start and End edit boxes are combined with drop-down lists. The lists contain the predefined positions, such as file begin, and and current edit point, and also markers and regions' boundaries.

You can type the begin of the selection into "Start" edit box, or select it from the drop-down list.

End of the selection can be typed into "End" edit box, or also selected from the drop-down list.

Or, if you wish, you can type the selection length to "Length" edit box.

"Time format" drop list chooses how the positions should be specified.
