Equializer command (Process menu)

Use this command to boost or attenuate selected frequency bands. Simple and graphic equalizer (up to 20 bands) modes are supported. Each band can be adjusted by +/-20 dB.

The following dialog box is shown:

The dialog can be resized by dragging on the border, to enlarge the graph window.

The frequency scale is logarithmic. This means the ratio between two adjacent frequency bands is the same.

Each band can be adjusted by dragging the small circle (fader) with the mouse up and down. Keyboard arrow keys can also be used. Left and right arrow keys switch to another band. Up and down arrows change the response by 1 dB at a time, in the selected band, which is shown with black circle.

Frequency response in the selected band edit box shows how much the sound will be boosted or suppressed. Each 6 dB correspond to two times amplitude change.

Zero phase shift checkbox selects whether the equalizer should have no phase distortion. The command then takes more time to execute.

Simple/Multiband buttons allow to select equalizer type. Multiband is shown above. The number of bands can be set from 3 to 20. Simple equalizer only allows Bass/Treble adjustments:

Reset all button returns all faders to zero attenuation.

Save As button allows to save the settings to a file.

Load button allows to load saved settings.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
