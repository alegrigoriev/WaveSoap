Filter command (Process menu)

Use this command to suppress all sound below or above specified frequency, or suppress one specified frequency.

The following dialog box is shown:

The graph shows the frequency response of the filter.

The frequency scale is logarithmic. This means the ratio between two adjacent dotted lines is the same.

The dialog can be resized by dragging on the border, to enlarge the graph window.

The filter consists of three components that can be enabled or disabled separately:

Highpass filter:

It suppresses frequencies below the specified passband.

Highpass filter is described by four parameters:

  • passband frequency, in Hz;
  • maximum transfer loss above passband frequency, in dB;
  • stopband frequency (lower than passband) in Hz;
  • minimum transfer loss on stopband frequency and below, in dB;

Lowpass filter:

It suppresses frequencies above the specified passband.

Lowpass filter is described by four parameters:

  • passband frequency, in Hz;
  • maximum transfer loss below passband frequency, in dB;
  • stopband frequency (higher than passband) in Hz;
  • minimum transfer loss on stopband frequency and above, in dB;

Notch filter:

It suppresses frequencies in the narrow stopband.

Notch filter is described by two parameters:

  • Center frequency, in Hz. The filter has theoretical zero transfer on this frequency;
  • passband width on -3dB transfer, in Hz. The graph shows lower boundary of the band.

Parameters of the filters can be set either by entering them in the edit boxes, or by using the mouse to dragg the reference points on the graph.

Each filter has two reference pointes. One is on the passband boundary, another is on the stopband.

You can drag them up and down, increasing or decreasing desired signal loss. You can drag them sideways, changing the boundary frequencies. Dragging a reference point while holding Ctrl key will change both frequencies at the same time, keeping constant ratio between them.

Keyboard can also be used. A reference point with input focus is shown as a small blinking circle. Left arrow and right arrow keys switch the focus to the next reference point. Shift+Left Arrow and Shift+Right Arrow keys move a reference point on the frequency axis. Ctrl+Left Arrow and Ctrl+Right Arrow keys move both reference points at the same time, keeping constant ratio between them. Up Arrow and  Down Arrow keys move the reference point by 1 dB at a time.

Highpass filter, Notch Filter and Lowpass Filter checkboxes enable the corresponding filters.

Passband transfer edit box specifies maximum loss for the selected filter, whose reference point is shown as a small black circle. Valid range is from -20 dB to 0. An edit box next to it specifies its corresponding frequency.

Stopband transfer edit box specifies minimum loss for the selected filter, whose reference point is shown as a small black circle. Valid range is from -120 dB to 0. An edit box next to it specifies its corresponding frequency.

Zero phase shift checkbox selects whether the filter should have no phase distortion. The command then takes more time to execute.

Reset button returns all filters to default parameters..

Save As button allows to save the settings to a file.

Load button allows to load saved settings.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
