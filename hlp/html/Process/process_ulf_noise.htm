Turntable Noise Reduction command (Process menu Sound Cleanup submenu)

Use this command to remove low-frequency static from the vinyl LP recording.

Shortcuts:

Toolbar:

Two components are considered: ultra-low frequency common mode noise ("horizontal wobble"), and differential mode noise caused by the disk warp ("vertical wobble"). The latter shows up as opposite waves in the left and right channels (the wave view is magnified):

As you can see, the wobble can reach about 3% of the full amplitude range.

The following dialog is opened as the menu command is selected:

Suppress "Vertical wobble" noise up to: check mark enables it, and the edit box specifies the frequency range to suppress. Allowed range is from 1 to 1000 Hz, default value is 200 Hz.

Suppress low frequencies up to: check mark enables it, and the edit box specifies the frequency range to suppress. Allowed range is from 1 to 1000 Hz, default value is 20 Hz.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
