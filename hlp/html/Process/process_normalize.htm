Normalize command (Process menu)

Use this command to bring the sound volume to the specified peak value.

The following dialog box is shown:

"Normalize to" slider and edit box allow to enter the desired peak volume. It can be specified in percents of the full range (allowed values are from 1 to 100), or in decibels, in range from -40 to 0. The drop list below the edit box selects percents or dB.

If "Equally adjust both channels" checkbox is checked, both channels volume will be multiplied by the same factor, but their peak volume may be different. Otherwise, the peak volume in each channel will be brought to the specified level, but it may require different multiplier for them.

Save Undo checkbox enables undoing the operation. If it's not checked, you cannot revert to the previous volume.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
