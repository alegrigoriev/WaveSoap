Click Removal command

(Process menu Sound Cleanup submenu)

Use this command to automatically remove click which are part of a vinyl LP record.

Shortcuts:

Toolbar:

Not all clicks can be reliably found and removed. It is recommended to manually clean the sound first from the largest clicks, using Interpolate command, and only then use automated Click Removal command.

The following dialog is opened as the menu command is selected:

The click detection algorithm uses the following parameters:

 Minimum click amplitude - minimum spike height to be considered a click. Valid range is from 50 to 5000, default is 200.

Click to noise ratio - how many times a click amplitude should be more than the filtered noise envelope. The sound is passed through a filter to suppress all data not click-alike. Valid range is from 1.5 to 20, default is 4.

Maximum click length - maximum length that the program will try to interpolate. Valid range is from 6 to 64, default is 32.

Noise attack rate - noise envelope filter constant, used when the noise amplitude increases. Valid range is from 0.001 to 0.99, default is 0.5. Higher values mean the program will accept sharp increases in high frequencies without taking them for a click, but is less likely to detect a click.

Click proximity factor - noise envelope filter constant, used when the noise amplitude decreases. Valid range is from 0.001 to 0.99, default is 0.01. Higher values mean the program will detect clicks that are closer, but false detection is more likely.

Import clicks from a file checkbox and an edit box - when checked, the clicks positions and lengths will be read from a text file, instead of running detection algorithm. The file name should be typed into an edit box, or selected with "Browse" button next to it.

Log all found clicks to a file checkbox and an edit box - when checked, the clicks positions and lengths will be written to a text file. The file name should be typed into an edit box, or selected with "Browse" button next to it. If "Log clicks only, don't change the wave" box is checked, the actual clicks won't be removed. Only detection will run.

Set defaults button reset all values to defaults.

Revert button resets all values to those loaded when the dialog was opened.

Save button saves the current settings to a text file. Load button loads previously saved settings.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
