DC offset command (Process menu)

Use this command to add a constant value to all the selected samples, to compensate for DC offset introduced during analog recording.

The following dialog box is shown:

WaveSoap can automatically detect DC offset, or an explicitly specified value can be used.

If "Automatically detect and remove" is selected, the necessary offset is calculated by averaging the samples. If "Compute DC offset from 5 seconds only", only the first 5.4 seconds worth of data is averaged. Such duration is chosen because it's exactly 3 rotations of an LP.

If "Adjust DC offset by" is selected, any desired value can be entered, valid range is from -32768 to 32767.

Save Undo checkbox enables undoing the operation. If it's not checked, you won't be able to undo the operation and all previous operations.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.
