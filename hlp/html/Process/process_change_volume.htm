Change Volume command (Process menu)

Use this command to change volume of the selected area or of the whole file.

The following dialog box is shown for a stereo file:

Lock checkbox forces the same volume change for both channels.

You can specify volume change either in percents or decibels. It can be selected by a drop-down list .

If percents are selected, 100% means no change, less than 100 means make the sound softer, more than 100 means make it louder. The allowed range is from 1% to 10000 percents.

If decibels (dB) are selected, 0 dB means no change, negative value makes the sound softer, positive value makes it louder. 6dB corresponds to about 2 times the amplitude change. The allowed range is from -40 to +40 dB.

Save Undo checkbox enables undoing the operation. If it's not checked, you cannot revert to the previous volume.

Selection button allows to specify range and channels to perform the operation. It opens a selection dialog.

For a mono file, the dialog box is different:

It doesn't contain Lock checkbox and edit box for the second channel.
