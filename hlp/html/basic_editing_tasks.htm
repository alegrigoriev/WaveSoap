Basic Editing Tasks

Not unlike text editing, audio work consists of capturing, moving and modifying sound data.

You can capture the data from an analog source (soundcard input) or from a CD.

You can delete parts of the file, copy part of the file to a temporary buffer (clipboard) or cut it, then paste the buffer to a different place in the same or different file.

You can change the sound volume in the whole file or in some part, or bring it to the desired peak level or mute some part of the file.
